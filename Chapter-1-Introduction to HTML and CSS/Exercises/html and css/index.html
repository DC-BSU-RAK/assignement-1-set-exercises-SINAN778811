HTML & CSS Quick Reference Guide

A handy guide with basic HTML tags and CSS properties.

HTML Tags

<h1> to <h6> - Headings

Used to define headings from most important (<h1>) to least (<h6>).

<h1>This is a Heading 1</h1>
<h2>This is a Heading 2</h2>

<p> - Paragraph

Defines a paragraph of text.

<p>This is a paragraph.</p>

<ul> and <li> - Unordered List

Creates a bullet-point list.

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol> and <li> - Ordered List

Creates a numbered list.

<ol>
  <li>First</li>
  <li>Second</li>
</ol>

<a> - Anchor (Link)

Creates a hyperlink to another page or site.

<a href="https://example.com">Visit Example</a>

<img> - Image

Embeds an image in the page.

<img src="image.jpg" alt="Sample Image">

<div> - Division

Defines a container for HTML content.

<div>Some grouped content here</div>

<span> - Inline Container

Used for styling small parts of text inline.

<p>This is <span style="color:red;">red text</span></p>

<code> and <pre> - Code Blocks

Used for displaying code with formatting.

<pre>
  <code>console.log("Hello World");</code>
</pre>

<link> - External Stylesheet

Links an external CSS file to the HTML.

<link rel="stylesheet" href="styles.css">

CSS Properties

color

Sets the text color.

p {
  color: darkblue;
}

font-family

Sets the font style of text.

body {
  font-family: 'Arial', sans-serif;
}

background-color

Changes the background color of an element.

header {
  background-color: #f0f0f0;
}

margin

Controls the space outside an element.

section {
  margin: 20px;
}

padding

Controls the space inside an element.

article {
  padding: 15px;
}

border

Applies a border around an element.

pre {
  border: 1px solid #ccc;
}

text-align

Aligns text (left, center, right).

h1 {
  text-align: center;
}

box-shadow

Adds shadow effects to elements.

.card {
  box-shadow: 2px 2px 8px rgba(0,0,0,0.1);
}
